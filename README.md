# Kuvika Store

## Resumen

Kuvika Store es una tienda online de joyas desarrollada en React, que implementa navegación entre distintas vistas mediante React Router. Los usuarios pueden:

- Explorar el catálogo completo de productos.
- Filtrar productos por categorías desde el menú.
- Visualizar el detalle de un producto.
- Agregar unidades al carrito desde la vista de detalle o desde el catálogo.
- Visualizar la cantidad de productos y el total en un carrito desplegable.

## Tecnologías utilizadas

- **Frontend:** React Router DOM para navegación
- **Lenguaje:** JavaScript (JSX)
- **Estilos:** CSS modularizado por componentes
- **Hooks:** useState, useEffect, useParams, useRef
